Bienvenido al repositorio del traductor definitivo. Este proyecto no es un simple script; es una **herramienta interactiva con interfaz gráfica** construida directamente sobre Google Colab, diseñada para traducir texto en tiempo real con la potencia de la API de Google Translate.

## ✨ Características Principales

* 🚀 **Traducción en Tiempo Real:** Olvídate de los botones. Escribe y la traducción aparecerá mágicamente al terminar tu frase.
* 🧠 **Detección Automática:** No necesitas decirle en qué idioma estás escribiendo; el sistema lo adivina instantáneamente.
* 🛡️ **Protección Anti-Bloqueos:** Lógica inteligente incorporada que procesa la traducción solo cuando detecta un espacio o signo de puntuación, evitando saturar la API.
* 🎨 **Interfaz Gráfica (GUI):** Construida con `ipywidgets` para una experiencia de usuario fluida, limpia y en dos pantallas simultáneas.

## ⚙️ Tecnologías Utilizadas

* **Python 3**
* **Googletrans** (`googletrans==4.0.0-rc1`)
* **Ipywidgets** (Para la magia visual)

## 🎮 Pruébalo tú mismo (Sin instalar nada)

No necesitas clonar el repositorio ni instalar Python en tu ordenador. Puedes ejecutar el entorno interactivo directamente en tu navegador haciendo clic en este botón:

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Sanchezx2/TraductorGoogleColab/blob/main/Traductor_Pro_Max.ipynb)

### Instrucciones de uso en Colab:
1. Abre el enlace de arriba.
2. Ve al menú superior y selecciona `Entorno de ejecución` -> `Ejecutar todas` (o presiona `Ctrl + F9`).
3. Baja hasta la última celda para ver la interfaz interactiva.
4. Escribe en la caja de la izquierda, pulsa ESPACIO, y disfruta de la magia.
